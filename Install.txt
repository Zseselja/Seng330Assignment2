Install.txt


1. setup eclipse C/C++
*I used the MinGW compiler when working on this project 
Make sure to and the proper path for your includes.
I used (C:\MinGW\mingw32\lib\gcc\mingw32\4.8.1\include\c++)

Refer to  this if you are having troubles: http://help.eclipse.org/mars/index.jsp?topic=%2Forg.eclipse.cdt.doc.isv%2Freference%2Fapi%2Forg%2Feclipse%2Fcdt%2Fcore%2Fdom%2Fast%2FExpansionOverlapsBoundaryException.html

2. Use git to Clone the this repo. Write in the git bash in the folder you want to clone the repo to. 
git clone https://github.com/Zseselja/Seng330Assignment2

3. Next you want to put the files in eclipse to build and run.
  - 1. Open a workplace one folder above where your local clone of the repository is.
  - 2. Create a new project and name it the same name as the repo. "Seng330Assignment2"
  - 3. The files should populate the Package Explorer. If not try hitting refresh.

4. Now that you have the files hit Ctrl + B to build all the files.

5. If a successful Hit Run aka "the green play button".

6. You should be able build Ship objects in run time and save them to file.
   Aswell load the contents of the already made ships into your program on
   after your first run.
